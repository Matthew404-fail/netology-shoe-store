# netology-shoe-store

## О приложении

Данный проект представляет собой интернет-магазин обуви и является дипломной работой в курсе "React" от Нетологии.

Приложение содержит следующие страницы:

1. Главная страница. [http://localhost:5173]
2. Каталог товаров. [http://localhost:5173/catalog]
3. О магазине. [http://localhost:5173/about]
4. Контакты. [http://localhost:5173/contacts]
5. Страница товара. [http://localhost:5173/catalog/21]
6. Корзина. [http://localhost:5173/cart]
7. 404 [http://localhost:5173/*]

Для хранения данных используется библиотека "redux-toolkit".
Для навигации по страницам приложения используется "react-router-dom"
Для обработки ввода пользовательских данных на странице 'Корзина' используется библиотека "react-hook-form"

## Запуск проекта

1. Выполнить установку необходимых зависимостей `npm i`
2. Для локального запуска проекта выполнить команду `npm run dev`
3. Запустить браузер, сайт будет доступен по адресу [http://localhost:5173]
